Вопрос 1: Что такое HTML?
HTML – это язык разметки, используемый для создания веб-страниц.
Вопрос 2: Что такое CSS?
CSS – это каскадные таблицы стилей, применяемые для оформления HTML-страниц.
Вопрос 3: Что такое JavaScript?
JavaScript – это язык программирования, который добавляет интерактивность на веб-сайты.
